# Faceless Video Generator

A modern web application for generating faceless videos using AI.

## What it does
Automatically generates complete faceless videos for content creators —
no camera needed. From script to final video using AI.

## 🚀 Getting Started

First, install dependencies:
npm install

Then, run the development server:
npm run dev

Open http://localhost:3000 with your browser to see the result.
